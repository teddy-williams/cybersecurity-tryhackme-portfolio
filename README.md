- Threat hunting investigations
- IOC extraction and validation
- Windows event log analysis

- Objective
- Tools used
- Workflow steps

- Findings (IOCs, evidence, timelines)
- Lessons learned
- Real-world mapping (SOC process / MITRE where relevant)
